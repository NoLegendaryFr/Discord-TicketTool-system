===========================================================================
                    🎟️  TICKET SYSTEM BOT  🎟️
===========================================================================

Un système de gestion de tickets robuste, performant et entièrement 
configurable pour Discord.js v14+. Ce bot inclut des fonctionnalités 
avancées de prise en charge (claim), de restriction de visibilité par 
rôles et de compteurs de tickets indépendants par catégorie.

🛠️  INSTALLATION & CONFIGURATION
---------------------------------------------------------------------------
1. CLONAGE : Téléchargez ou clonez le dépôt sur votre machine ou VPS.
2. DÉPENDANCES : Installez les modules nécessaires avec la commande :
   > npm install
3. CONFIGURATION : Remplissez votre fichier 'configTickets.json' avec 
   vos IDs (Client, Guild, Salons et Rôles).
4. LANCEMENT : Démarrez le bot en utilisant :
   > node app

---------------------------------------------------------------------------
🚀  FONCTIONNALITÉS
---------------------------------------------------------------------------
* Système de catégories : Types de tickets illimités (Support, Plainte...).
* Compteurs par catégorie : Numérotation propre (ex: mod-001, tech-001).
* Commande /claim : Prise en charge et renommage dynamique du ticket.
* Commande /restreindre : Verrouillage de la visibilité à un grade précis.
* Permissions dynamiques : Gestion automatique via le fichier de config.
